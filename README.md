# IntelMacOS_Optimisation

> **Bringing a new life to Intel Macs.**

## Overview

Intel-based Macs are still powerful machines, but Apple's increasing focus on Apple Silicon means they receive fewer optimisations and updates over time. **IntelMacOS_Optimisation** is a project dedicated to keeping Intel Macs fast, responsive, and relevant for as long as possible.

Whether you're looking to squeeze more performance out of your machine, reduce unnecessary resource usage, or simply keep your Intel Mac feeling fresh — this project has you covered.

## Roadmap

| Phase | Description | Status |
|-------|-------------|--------|
| 🚀 Phase 1 | Optimisation **shell scripts** — easy to run, no installation required | 🔨 In Progress |
| 📦 Phase 2 | Packaged **macOS app** — a polished GUI wrapping all the scripts for a seamless experience | 🗓 Planned |

### Phase 1 — Script

The first iteration of this project is a collection of shell scripts that you can run directly on your Intel Mac. They target common bottlenecks and settings that can meaningfully improve day-to-day performance.

**Goals:**
- Optimise system preferences for performance
- Clean up junk files and free up disk space
- Manage startup items and background processes
- Improve battery life on Intel MacBook models

### Phase 2 — App

Once the scripts are mature and well-tested, they will be wrapped into a native macOS application with a clean, simple interface — no Terminal required.

**Goals:**
- One-click optimisation
- Scheduled maintenance tasks
- Visual performance dashboard
